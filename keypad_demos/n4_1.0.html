How to Use:

Button Values:

  • Button 1: [1, 2, 3]
  • Button 2: [4, 5, 6]
  • Button 3: [7, 8, 9]
  • Button 4: [0, *, #]

Example: To enter "3018", press:

  • Button 1 three times (for 3)
  • Button 4 once (for 0)
  • Button 1 once (for 1)
  • Button 3 twice (for 8)

Press CONFIRM to enter the current value into the PIN.

Press CANCEL to reset the current input.
